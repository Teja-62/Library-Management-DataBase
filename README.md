## Project Title
Library Management Database System

## Description
This project is a simple Library Management System developed using SQL. It stores information about books, students, and issued books. The project demonstrates basic database operations such as table creation, data insertion, and retrieval using SQL queries.

## Database Name
library_db

## Tables Used

### 1. Books
Stores book information.
- book_id
- title
- author
- available_copies

### 2. Students
Stores student information.
- student_id
- student_name

### 3. Issued_Books
Stores details of books issued to students.
- issue_id
- student_id
- book_id
- issue_date

## Features
- Add and manage books
- Add and manage students
- Record book issue details
- View all books
- View all students
- View issued books
- Display issued book details using JOIN operations

## Technologies Used
- SQL
- MySQL

## SQL Concepts Demonstrated
- CREATE DATABASE
- CREATE TABLE
- PRIMARY KEY
- FOREIGN KEY
- INSERT INTO
- SELECT Queries
- JOIN Operations

## Outcome
This project helps in understanding the fundamentals of relational databases and SQL operations through a simple Library Management System.
